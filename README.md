# Trivializer
React Reading List App with simple Auth

## Technologies Used: 
<ul>
	<li>React</li>
  <li>Node.js</li>
	<li>JSX</li>
  <li>MongoDB</li>
  <li>HTML</li>
  <li>CSS</li>
  <li>React-Bootstrap</li>
  <li>NPM (Pusher, chart.js, axios, express, mongoose, morgan, passport, node-qrcode, react-response, react-scroll)</li>
</ul>
